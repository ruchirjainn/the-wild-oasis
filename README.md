**Installation**
1) Set up supabase database
In src -> services -> supabase.js, replace ur own supabaseUrl & supabaseKey

2) Clone the repository
- https://github.com/ruchirjainn/the-wild-oasis.git

3) Redirect to the directory
- cd the-wild-oasis

4) Install npm packages
- npm install OR npm i

5) start the project
- npm run dev
